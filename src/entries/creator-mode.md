# Summary 

Here is how to enable creator mode.

## Example Usage {.pt-4}

### Markdown:

```md settings:code:hide-clipboard
~~~ts settings:code:hide-clipboard highlight:[3-5] settings:code:creator-mode
export default function Page() {
  return (
    <div className="text-slate-200">
      Hello world
    </div>
  );
}
~~~
```

### Result:

```ts settings:code:hide-clipboard highlight:[3-5] settings:code:creator-mode
export default function Page() {
  return (
    <div className="text-slate-200">
      Hello world
    </div>
  );
}
```
